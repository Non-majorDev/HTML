태그란?

태그?

태그는 단순하게 이야기하자면 명령어와 같다. 열린태그와 닫힌태그로 구분된다.

열린태그 / 닫힌태그

열린태그와 닫힌태그 모두 존재하는 태그와 열린태그만 존재하는 태그가 존재한다.


>> 위와 같이 태그들 사이에 내용을 입력한다.


h1~h6, strong 과 같이 열린태그, 닫힌태그 모두 존재하는 경우(주로 본문의 내용을 입력하는 경우)는 열린태그와 닫힌태그 사이에 내용을 입력해준다.

열린태그는 <> 기호 안에 해당 태그를, 닫힌태그는 <> 기호 안에 /기호를 더한 /태그 의 형태를 입력하면 된다.

열린태그만을 사용하는 태그

줄바꿈 태그인 br과 같이 열린태그만을 통해 표현되는 태그들도 존재한다.
>> 이 경우에는 열린태그만을 사용한다.

줄바꿈을 하지 말고, p 태그를 통해 단락을 나누어보자!

물론 줄바꿈을 통해 단락을 구분지어도 좋지만, 더욱 프로그래머답게 구조를 정리하는 것은 어떨까?

이 때 사용하는 태그가 바로 p 태그이다.



img 태그의 src 뒤에 붙은 부분과 width 뒤에 붙은 부분 태그의 속성으로 뒤에서 배울 부분이니 우선 넘어가도록 하자.

위 코드에서 p의 열린태그 뒤쪽에 있는 style="margin-top:100px;" < 이 부분이 CSS를 통해 p 태그의 간격을 조절한 것이다. 해당 부분은 나중에 CSS를 배우며 더 자세히 다루도록 하고, CSS를 통해 간격, 글자 크기, 글자 색 등 여러가지 기능을 할 수 있다는 것만 알아두고 넘어가도록 하자.

이처럼 줄바꿈 태그인 br을 사용해서 단락을 나누는 것 보다 조금 더 체계적으로 단락을 묶어주는 태그인 p 태그를 배워보았다.
