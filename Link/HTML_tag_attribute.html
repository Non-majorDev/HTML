태그의 속성

속성을 이용하여 이미지파일을 넣어보자.

만약 웹페이지에 글자만 적혀있다면 삭막하지 않을까?
우리가 지금부터 할 것은 삭막한 웹페이지에 이미지를 넣어서 활기를 불어 넣어 볼 것이다.

우선 이미지를 넣는 태그는 image 의 줄임말인 img 이다. img 태그는 태그의 이름만으로는 사용할 수 없어 정보를 표기하기 위한 문법을 만들었는데, 이것이 바로 속성(attribute)이다.

img 태그의 속성은 여러가지가 있는데, 전부 설명하기엔 당연히 무리이므로, 필요한 속성들이 있다면, 검색으로 추가하면 될 것이고, 한번 이전에 태그를 배울때 보았던 코드 중 이미지가 어떻게 삽입되었는지 해당 코드를 보며 이해해보자.




src 라고 적힌 속성은 source의 줄임말로, 이미지의 주소를 붙여넣으면 된다. 이미지의 주소는 인터넷의 이미지 주소를 넣어도 되고, 직접 내 파일에 있는 이미지를 넣어도 된다.

위 사진의 경우는 해당 html 문서가 있는 경로를 기준으로 상위 폴더에 접근 > Image 폴더에 접근하여 img_tag_attribute 폴더에 접근, img_tag_attribute_1.png 파일에 접근하였다는 코드이다.

다음으로 width 속성을 보면, 간단하다. width 속성은 해당 인터넷 창에 얼마나 크게 나타내어 질 것인지를 나타내는 코드이다.
